Excel Import

Import your bank transactions from Excel files

Supported formats: .xlsx, .xls, .csv

Currently only Yapı Kredi is supported

Important Notice

Import operation cannot be undone. Transactions will be created in "Import" category with bank tag.

How to Use?

1

Login to your Yapı Kredi Internet Banking

2

Go to Account Summary menu

3

Select date range and click "Download Excel"

4

Upload the downloaded Excel file here

5

Transactions will be automatically created in "Import" category

6

You can edit categories and tags later
