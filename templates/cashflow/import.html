Excel Import

Import your bank transactions from Excel files

Supported formats: .xlsx, .xls, .csv

Supported banks: Yapı Kredi, Kuveyt Türk

After Import

  • • Transactions will be created in "Import" category with your bank's tag
  • • You can edit categories and tags later from the main page
  • • Import operation cannot be undone, please verify your file before uploading

How to Export Excel from Your Bank?

Yapı Kredi

1

Go to Yapı Kredi Internet Banking website and login

2

Select your credit card from "My Credit Cards"

3

Open "Account Statement" page and click "Export to Excel"

4

Upload the downloaded Excel file using the form above

Kuveyt Türk

1

Go to Kuveyt Türk Internet Banking website and login

2

Click "Cards" tab, then go to "My Cards"

3

On "All Statements" page, select the statement you want

4

Click "Print", close the print dialog, then copy the table from popup to Excel

5

Save as Excel file and upload it using the form above
